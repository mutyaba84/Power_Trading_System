POWER TRADING SYSTEM - START HERE

1. Double-click INSTALL_FIRST.bat.
2. Open .env and add your Alpaca API keys.
3. Double-click start_project.bat.
4. Choose Paper Trading first.
5. To stop everything, double-click stop_project.bat.

Included:
- INSTALL_FIRST.bat installs dependencies.
- start_project.bat opens the control center.
- backend_watchdog.bat restarts backend if it crashes.
- health_check.bat checks system status.
- stop_project.bat stops backend/frontend.
- update_system.bat updates dependencies and pulls Git updates if available.

Never share your real .env file.
Use paper mode before live trading.
