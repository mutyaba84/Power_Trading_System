POWER TRADING SYSTEM - START HERE

1. Put these files in the main project folder:
   - INSTALL_FIRST.bat
   - start_project.bat
   - stop_project.bat
   - .env.example

2. Double-click:
   INSTALL_FIRST.bat

3. Open .env and add your Alpaca keys.

4. Double-click:
   start_project.bat

5. To stop everything, double-click:
   stop_project.bat

IMPORTANT:
- Do not share your real .env file.
- Use Alpaca paper mode first.
- Keep backend and frontend terminal windows open while running the system.
